Disclaimer & Technical Notes

This DNS Speed Test tool runs entirely in your browser, sending live DNS-over-HTTPS (DoH) queries from your local environment. No server-side code is involved, and no personal data is collected.

Time Measurement & Browser Overhead: We use the browser's fetch and performance.now() APIs to measure full DoH round-trip times. Results include HTTP/TLS overhead.

Timeouts & Unavailable Results: Each query times out after 5 seconds. If a server does not respond, it appears as Unavailable.

Influence of Local Conditions: Network latency, bandwidth, and browser load can cause variability. Run multiple tests for better accuracy.

Results are indicative. Use them as a guide for relative comparison under your specific network conditions.
