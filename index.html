Technical Information

Browser-Based Testing

All tests run entirely in your browser using DNS-over-HTTPS (DoH) queries. No server-side processing, ensuring privacy and real-world results.

Performance Metrics

Measures full DoH round-trip times including HTTP/TLS overhead. Use results for relative comparison between DNS servers.

Timeout Protection

Each query times out after 5 seconds to ensure responsive testing. Servers showing "Unavailable" are either slow or unreachable.

Network Conditions

Results vary based on your current network, bandwidth, and geographic location. Run multiple tests for accurate comparison.

Connection Caching

Implements intelligent caching to reuse warm connections and provide more accurate results for subsequent queries.

No Data Collection

Zero personal data collection. All testing happens locally in your browser without sending information to external servers.
